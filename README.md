# Disable Admin Bar Hover

WordPressのアドミンバーのトップセカンダリメニュー（`#wp-admin-bar-top-secondary`）のマウスオーバー機能を無効化するプラグインです。

## 概要

このプラグインは、WordPressのアドミンバーの右側にあるメニュー項目にマウスオーバーしたときに表示されるドロップダウンメニューを無効化します。これにより、誤ってメニューを開いてしまうことを防ぎ、よりシンプルなユーザー体験を提供します。

## 機能

- **マウスオーバー時のサブメニュー非表示**: メニュー項目にカーソルを合わせてもサブメニューが表示されません
- **ホバー効果の無効化**: マウスオーバー時の背景色変更などの視覚効果を無効化
- **リンクのクリックは有効**: メニュー項目自体のクリック機能は維持されます
- **管理画面・フロントエンド両対応**: アドミンバーが表示されるすべての場所で動作します
- **軽量実装**: CSSのみで実装されているため、パフォーマンスへの影響がありません

## インストール

### 手動インストール

1. このプラグインフォルダ全体を `/wp-content/plugins/` ディレクトリにアップロード
2. WordPressの管理画面で「プラグイン」メニューからプラグインを有効化

### 有効化

1. WordPress管理画面にログイン
2. **プラグイン** → **インストール済みプラグイン** に移動
3. **Disable Admin Bar Hover** を見つけて「有効化」をクリック

## 使い方

プラグインを有効化するだけで自動的に機能します。特別な設定は必要ありません。

有効化後、アドミンバーの右側のメニュー項目（例：ユーザー名、サイト名など）にマウスオーバーしても、サブメニューが表示されなくなります。

## 動作要件

- WordPress 4.0 以上
- PHP 5.6 以上

## 対象となるメニュー

このプラグインは、以下のような `#wp-admin-bar-top-secondary` に含まれるメニュー項目に影響します：

- ユーザーアカウントメニュー
- サイト切り替えメニュー（マルチサイトの場合）
- その他のセカンダリメニュー項目

## アンインストール

1. WordPress管理画面の「プラグイン」メニューから「Disable Admin Bar Hover」を無効化
2. 「削除」をクリックしてプラグインを完全に削除

## 技術詳細

このプラグインは、以下のCSSルールを適用することで機能を実現しています：

- サブメニューの `display: none` による非表示化
- 親メニューの `pointer-events: none` によるホバーイベントの無効化
- リンク自体の `pointer-events: auto` によるクリック機能の維持
